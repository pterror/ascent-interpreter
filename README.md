# ascent-interpreter

Interpreter and JIT compiler for [Ascent](https://github.com/s-arash/ascent) Datalog programs.

## Overview

Ascent is a Datalog DSL embedded in Rust that compiles rules at `rustc` time via macros. This project provides an alternative execution model: runtime interpretation and JIT compilation without requiring Rust compilation.
